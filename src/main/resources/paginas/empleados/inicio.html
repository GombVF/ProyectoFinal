La sección de Pedidos es una sección pensada para que usted, el cliente, tenga un área donde consultar sus operaciones relacionadas a sus pedidos, las consultas de sus pedidos pendientes y finalizados, así como también darle un espacio para que pueda solicitar nuevos pedidos los cuales serán enviados a un domicilio que tenga relacionado.
La sección de Servicios es una sección pensada para que usted, el cliente, tenga un área donde consultar sus operaciones relacionadas a sus servicios, las consultas de sus servicios pendientes y finalizados, así como también darle un espacio para que pueda solicitar nuevos servicios los cuiales serán realizados para la máquina que usted indique.
La sección de Mi Domicilio es una sección pensada para que usted, el cliente, tenga un área para mantener actualizada su información, en este caso, podrá dar de alta un domicilios, actualizarlo o incluso darlo de baja.

Es importante destacar que es necesario que cuente con un domicilio si quiere solicitar un pedido, ya que el sistema obtiene su domicilio para dar de alta el pedido y determinar a qué dirección mandar el pedido.

La sección de Mis Máquinas es una sección pensada para que usted, el cliente, tenga un área para administrar sus máquinas, de manera que podrá registrar una nueva máquina que adquiera, actualizar una máquina previamente registrada o eliminar una máquina debido a que ya no se encuentra en operación.
Para solicitar un pedido es necesario registrar un domicilio.
Para solicitar un servicio es necesario tener una máquina registrada.
